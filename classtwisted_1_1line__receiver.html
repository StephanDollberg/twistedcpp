Detailed Description

template<typename ChildProtocol>
class twisted::line_receiver< ChildProtocol >

Protocol to receive lines delimited by a certain delimiter.

Provided callbacks:

void line_received(const_buffer_iterator begin, const_buffer_iterator end);

Will be called whenever a line delimited by the delimiter has been received. [begin, end) is the data range. The range does not include the delimiter.
